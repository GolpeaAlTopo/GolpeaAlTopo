# GolpeaAlTopo

¡Bienvenido a **GolpeaAlTopo**! Un juego interactivo desarrollado con **Angular 21** donde pondrás a prueba tus reflejos.

## Repositorio
Puedes jugar a la versión más reciente aquí:
[https://golpeaaltopo.github.io/GolpeaAlTopo/](https://golpeaaltopo.github.io/GolpeaAlTopo/)

## Requisitos Previos

Antes de empezar, es necesario tener instalado:
Node.js: Versión 22.x o superior (recomendado).
npm: Incluido con Node.js.
Angular CLI: Se instala globalmente con `npm install -g @angular/cli`.

## Instalación en Local

Para correr el proyecto de forma local:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/GolpeaAlTopo/GolpeaAlTopo.git
   cd GolpeaAlTopo
   ```

2. **Instalar las dependencias:**

```bash
npm install
```

3. **Iniciar el servidor**

```bash
ng serve
```

4. **Abrir el navegador en http://localhost:4200/**
